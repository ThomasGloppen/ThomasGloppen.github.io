Datamaskinens oppbygging og virkemåte

Prosessor

Intel

Amerikansk produsent av prossorer
Grunnlagt 1968 - laget intel 4004 CPU, 4bit cpu.
Dagens CPUer er 64bit. 4bit gir 2^4 som største tall = 16
For 5-6 år siden var standard størrelse 32bit (4294967296)
Dagens CPU kan jobbe med tall på 2^64 = 18446744073709551616

AMD

AMD (Advanced Micro Devices) laget sin første CPU 8080 AM9080 (en klone av intel 8080).

ARM

Acorn (Advanced) RISC (Reduced Instruction Set Computer) Machine.
RISC CPUer har få og nkle instruksjoner (i motsetning til Intel)
Hver instruksjon skal ta minst mulig tid.
Kompliserte beregninger består av mange instruksjoner.
Brukes i mobiler (nesten alle?)

Prosessoren er hjernen i datamaskinen.
Hastigheten på en CPU er mellom 2 - 4 GHz for nye maskiner.

Sammenligning av en 8-bit og en 64-bit CPU

En 8-bit CPU kan jobbe med tall på 2^8 = 256
En 64-bit CPU kan jobbe med tall på 2^64 = 18446744073709551616
